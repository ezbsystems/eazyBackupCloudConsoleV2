Section 01

Colour Tokens

The dark palette is built on a cool blue-black base (not pure black) with four complementary semantic colour families extending the brand orange axis.

🎨
Brand orange #FE5000 is restricted. In the dark theme it appears as: accent dots, active sidebar icons, small status indicators, the "Popular" pricing badge, and inline links. The primary button colour is #d55d1d (softer). Never use #FE5000 as a fill on large surfaces.

Section 09

Buttons — All Variants

11 button variants covering every action type in the client area. Use semantic variants intentionally — danger for destructive, success for confirm/approve, warning for caution actions.

Core Variants

Primary — main save/submit/create actions
Secondary — cancel, back, neutral actions
Outline — tertiary actions, secondary CTAs
Ghost — low-priority, inline actions
Orange Outline — orange-tinted actions (upgrade nudge, feature callout)

Semantic Variants

Success — confirm, approve, activate, enable
Danger — delete, remove, revoke, disconnect
Warning — caution actions, pause, suspend
Info — informational, run, sync, refresh
